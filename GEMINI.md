# 🚀 Portfolio de Jordan Aziaha - BTS SIO SISR

Ce fichier contient les informations essentielles sur le projet de portfolio conçu pour le BTS SIO option SISR (Solutions d'Infrastructure, Systèmes et Réseaux).

## 📌 Présentation du Projet
Ce site est un portfolio "One Page" moderne, conçu pour présenter le parcours, les compétences techniques et les réalisations de **Jordan Aziaha**, futur expert en infrastructure IT et IA.

- **Étudiant :** Jordan Aziaha (2ème année BTS SISR)
- **Objectif :** Validation des compétences E4 (Support et mise à disposition de services informatiques) et E6 (Parcours de professionnalisation).
- **Spécialité :** Infrastructure, Réseaux, Systèmes et Automatisation IA.

## 🛠 Stack Technique
Le portfolio est bâti sur une architecture moderne et performante :
- **Framework :** [Next.js 15+](https://nextjs.org/) (React 19)
- **Styling :** [Tailwind CSS](https://tailwindcss.com/)
- **Icônes :** [Lucide React](https://lucide.dev/)
- **Animations :** CSS pur et Framer Motion (si applicable)
- **Langage :** TypeScript

## 📂 Structure du Portfolio
Le site est divisé en sections stratégiques pour le BTS :
1.  **Hero :** Accroche visuelle et présentation rapide.
2.  **À propos :** Parcours (spécialités NSI/SES) et vision professionnelle (Entrepreneur IA).
3.  **Projets :**
    - **Projet GSB** (Projet académique de 1ère année - Gros projet).
    - **Stage de 1ère année** (Ministère de la justice).
    - **Alternance Automatisation IA** (Création de solutions IA, adaptation au référentiel SISR requise).
    - **Projet Client Agence IA** (Déploiement VPS Hostinger, Nginx, Pare-feu, n8n, Chatwoot).
    - **Projet OPNsense sur VMware** (Petite infrastructure réseau et sécurité - Projet à venir).
4.  **Compétences :** Systèmes (Windows/Linux), Réseaux, Virtualisation, Automatisation, Sécurité, Innovation IA.
5.  **Veille Technologique :** Focus sur l'IA comme levier de transformation business.
6.  **Documents :** Zone de téléchargement du CV et du Tableau de Synthèse.
7.  **Contact :** Formulaire de contact et réseaux sociaux.

## 🚀 Comment lancer le projet localement ?

1.  **Installer les dépendances :**
    ```bash
    npm install
    ```
2.  **Lancer le serveur de développement :**
    ```bash
    npm run dev
    ```
3.  **Accéder au site :** Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.
