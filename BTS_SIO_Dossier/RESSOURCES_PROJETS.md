## 4️⃣ Projet Perso / Client (Agence Automatisation IA)
* **Contexte :** Mission concrète de déploiement d'une infrastructure pour un client.
* **Documentation détaillée :** Voir le fichier `DOC_PROJET_VPS_IA.md` dans ce dossier.
* **Technologies :** 
  - Serveur : VPS Hostinger (Linux)
  - Serveur Web / Reverse Proxy : Nginx
  - Sécurité : Configuration du pare-feu (UFW / iptables)
  - Applications hébergées : n8n (automatisation), Chatwoot (support client omnicanal)
* **Cible SISR :** Excellent projet pour l'épreuve E4 ! Il valide l'installation, la configuration d'un service (web, applicatif) et la sécurisation.
